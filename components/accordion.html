Accordion

This component requires the accordion.plugin.js file. Within the .accordion class each accordion item requires a div wrapper. Within the div wrapper there is a .header class and a .content class. The .header class is the clickable element that will reveal what is hidden in the .content class. By default .content is hidden by javascript just in case javascript is not available.

Emmet

.accordion>div>(.header{header}+.content>p{lorem ipsum})

Plugin Options

Option Type Default Description
hiddenOnLoad Boolean true Hide contents on Load
singleOpen Boolean true Only open one at a time.

JS Example

$(function () {
        	var $accordion = $('.accordion');
        	$accordion.accordion({
        		hiddenOnLoad: true,
        		singleOpen: true
        	});
        });
